### 2.1 问题1 模型：一维非稳态热传导模型（Crank-Nicolson 有限差分求解）

#### 2.1.1 模型名称

**基于分段热力学参数的一维 Crank-Nicolson 热传导模型**

#### 2.1.2 创新方向

✅ **多模型融合组合创新** — 将经典热传导 PDE、分段参数策略、Crank-Nicolson 隐式格式与 PSO 参数估计相融合

#### 2.1.3 五大核心内容

##### (1) 核心原理

焊接区域厚度（0.15 mm）远小于其长宽尺寸，仅考虑厚度方向（z 方向）的一维热传导：

$$\frac{\partial T}{\partial t} = a^2 \frac{\partial^2 T}{\partial z^2}, \quad 0 < z < l, \quad t > 0$$

- **初始条件**：$T(z, 0) = 25°\text{C}$（车间环境温度）
- **边界条件（对流换热，牛顿冷却定律）**：

$$-\left.\frac{\partial T}{\partial z}\right|_{z=0} = h[T_{air}(x) - T(0,t)]$$

$$\left.\frac{\partial T}{\partial z}\right|_{z=l} = h[T_{air}(x) - T(l,t)]$$

其中导热系数已归入无量纲换热系数 $h = h_1/k$。

**炉内环境温度** $T_{air}(x)$ 由各温区设定温度按分段线性函数构造：

$$T_{air}(x) = \begin{cases}
\frac{T_1-25}{25}x + 25, & 0 \leq x < 25 \\
T_1, & 25 \leq x < 197.5 \\
\frac{T_6-T_1}{5}(x-197.5) + T_1, & 197.5 \leq x < 202.5 \\
T_6, & 202.5 \leq x < 233 \\
\frac{T_7-T_6}{5}(x-233) + T_6, & 233 \leq x < 238 \\
T_7, & 238 \leq x < 268.5 \\
\frac{T_8-T_7}{5}(x-268.5) + T_7, & 268.5 \leq x < 273.5 \\
T_8, & 273.5 \leq x < 339.5 \\
\frac{25-T_8}{5}(x-339.5) + T_8, & 339.5 \leq x < 344.5 \\
25, & x \geq 344.5
\end{cases}$$

##### (2) 情境适用性

- 适用条件：PCB 焊接区域可近似为一维薄板，回焊炉内以强制对流为主要传热方式
- 数据匹配：利用附件实验数据（v=70 cm/min, 特定温度设定）拟合参数，使模型与实际工艺匹配
- 目标适配：能够计算任意位置的温度、升降温速率、恒温时间等制程指标

##### (3) 真实创新点

| 对比维度 | 传统方法 | 本模型创新 |
|:---|------|------|
| 参数处理 | 全域统一热力学参数 | **5段分区参数**，根据炉内温度区域自适应切换 |
| 数值格式 | 显式 Euler 格式（条件稳定） | **Crank-Nicolson 隐式格式**（二阶精度 + 无条件稳定） |
| 边界处理 | Dirichlet 恒温边界 | **对流换热边界**（更接近实际物理过程） |
| 参数获取 | 查表或经验值 | **PSO 全局优化** 基于实验数据反演参数 |
| 求解效率 | 逐时间步求解线性方程组 | **预分解矩阵** C = A⁻¹B，每步仅需矩阵-向量乘法 |

##### (4) 模型局限性 + 改进方案

| 局限性 | 误差来源 | 改进方案 | 敏感性分析 |
|--------|--------|--------|--------|
| 一维假设忽略横向传热 | PCB 边缘温度梯度 | 建立二维/三维模型 | 边缘效应影响 < 约3% |
| 忽略辐射传热 | 高温区（>200°C）辐射显著 | 增加 Stefan-Boltzmann 辐射项 | 高温区影响 ≈ 5~8% |
| 分段常数物性 | 材料性质连续变化 | 采用温度相关的变物性模型 | 热扩散系数 ±10% → 峰值温度偏移 ±3°C |
| 参数估计依赖实验数据 | 实验误差传递 | 增加多组实验交叉验证 | PSO 收敛到局部最优的风险 |
| 忽略传送带振动 | PCB 位置偏差 | 增加位置不确定性分析 | 对整体曲线影响 < 1% |

##### (5) 建模过程

```
[物理抽象]
   PCB 焊接区域 → 一维薄板 (0.15 mm 厚)
   炉内传热 → 强制对流主导
        ↓
[变量定义]
   空间变量: z ∈ [0, l] (厚度方向)
   时间变量: t ∈ [0, L/v] (过炉时间)
   中间变量: x = v·t (炉内位置)
   待估参数: a² (热扩散系数), h (无量纲换热系数)
        ↓
[模型构建]
   热传导 PDE + 对流边界条件 → Crank-Nicolson 差分格式
   三对角矩阵 A·T^{j+1} = B·T^j + c
        ↓
[数据适配]
   附件实验数据 → PSO 全局优化 → 拟合 10 个热力学参数
        ↓
[结果验证]
   对比模拟温度与实测温度 → R² / MSE 评估
```

### 2.2 问题2 模型：二分搜索约束满足模型

#### 2.2.1 模型名称

**基于二分搜索的传送带速度可行域求解模型**

#### 2.2.2 五大核心内容

##### (1) 核心原理

传送带速度 v 与炉温曲线指标之间存在单调关系：速度越大，加热时间越短，峰值温度越低，>217°C 时间越短。因此，满足制程界限的速度集合构成一个区间 [v_min, v_max]。寻找最大速度即为在单调可行域上搜索边界值。

**二分搜索策略**：
1. 确定搜索区间 [v_lo, v_hi] = [65, 100] cm/min
2. 计算中点 v_mid = (v_lo + v_hi) / 2
3. 若 v_mid 可行 → v_lo = v_mid（向右搜索）
4. 若 v_mid 不可行 → v_hi = v_mid（向左搜索）
5. 重复至 |v_hi - v_lo| < ε = 10⁻⁴

##### (2) 情境适用性

- 速度-温度指标具有单调性（已验证：65~100 cm/min 扫描）
- 约束条件明确且可量化（4 条制程界限）
- 搜索区间确定（制程界限自然界定上下界）

##### (3) 真实创新点

- 利用**单调性保证**将约束满足问题转化为二分搜索，避免暴力扫描
- 收敛速度：每次迭代区间减半，log₂(35/10⁻⁴) ≈ 19 次即可收敛
- 相比等步长扫描（步长 0.1 需 350 次），效率提升约 18 倍

##### (4) 模型局限性 + 改进方案

| 局限性 | 改进方案 |
|--------|--------|
| 依赖单调性假设 | 若不单调则改用扫描 + 黄金分割搜索 |
| 仅搜索单一可行边界 | 可扩展为搜索完整可行域 [v_min, v_max] |
| 二分搜索对噪声敏感 | 增加约束容忍度 ε_constraint |

##### (5) 建模过程

```
[确定约束集] → 4 条制程界限
       ↓
[单调性验证] → 65:5:100 cm/min 速度扫描
       ↓
[二分搜索] → 初始区间 [65, 100]
       ↓
[收敛判断] → |v_hi - v_lo| < 1e-4
       ↓
[结果验证] → 检查 v_max 处所有约束
```

### 2.3 问题3 模型：最小化面积的参数优化模型

#### 2.3.1 模型名称

**基于粒子群优化（PSO）的炉温曲线面积最小化模型**

#### 2.3.2 五大核心内容

##### (1) 核心原理

**优化问题数学表述**：

$$\min_{\mathbf{x}} \quad S(\mathbf{x}) = \int_{t_1}^{t_{peak}} (T(t; \mathbf{x}) - 217) \, dt$$

$$\text{s.t.} \quad 240 \leq T_{max} \leq 250$$
$$\quad 60 \leq t_{150-190} \leq 120$$
$$\quad 40 \leq t_{>217} \leq 90$$
$$\quad |dT/dt|_{max} \leq 3$$
$$\quad \mathbf{x} = [T_{1-5}, T_6, T_7, T_{8-9}, v]^T$$
$$\quad \mathbf{x}_{lb} \leq \mathbf{x} \leq \mathbf{x}_{ub}$$

**求解策略**：

- **方法一（推荐）**：采用**带自然选择的粒子群优化算法（SelPSO）**，在 5 维决策空间中进行全局搜索。粒子群规模 40~60，迭代次数 500~1000，惯性权重 0.8。PSO 通过粒子群的社会行为模拟实现高效全局优化，自然选择机制进一步避免陷入局部最优。
- **方法二（备选）**：两阶段网格搜索。第一阶段粗搜索（每维 5~7 节点），第二阶段在粗最优邻域内精细搜索（每维 7~11 节点）。

##### (2) 情境适用性

- 决策变量维度低（5 维），网格搜索可行
- 目标函数计算廉价（单次求解约 0.01s）
- 变量范围由设备能力自然界定

##### (3) 真实创新点

| 对比维度 | 传统方法 | 本模型创新 |
|:---|------|------|
| 优化算法 | 枚举/网格搜索 | **SelPSO 全局优化**（带自然选择机制的粒子群算法） |
| 参数处理 | 全域统一参数 | **5段分区参数**，根据炉内温度区域自适应切换 |
| 数值格式 | 显式 Euler 格式（条件稳定） | **Crank-Nicolson 隐式格式**（二阶精度 + 无条件稳定） |

- SelPSO 通过自然选择机制淘汰适应度差的粒子，相比标准 PSO 收敛更快且更不易陷入局部最优
- 决策空间维度低（5维），PSO 搜索充分，计算开销可控（单次求解约0.01s，1000次迭代约10s）

##### (4) 模型局限性 + 改进方案

| 局限性 | 改进方案 |
|--------|--------|
| 网格搜索在高维空间效率低 | 若扩展变量维度，改用 PSO / 遗传算法 |
| 离散搜索可能错过真正最优 | 精搜索后加梯度下降微调 |
| 目标函数仅考虑面积 | 可加权加入峰值温度偏差等指标 |

##### (5) 建模过程

```
[确定决策变量] → 5 个变量 + 取值范围
       ↓
[粗网格搜索] → 5×5×5×5×7 = 4375 组合
       ↓
[找到粗最优] → 确定精搜索邻域
       ↓
[精网格搜索] → 7×7×7×7×7 = 16807 组合
       ↓
[最优解输出] → 温区温度 + 速度 + 最小面积
```

### 2.4 问题4 模型：带对称性约束的多目标优化模型

#### 2.4.1 模型名称

**基于 PSO 双目标优化的炉温曲线对称性优化模型**

#### 2.4.2 五大核心内容

##### (1) 核心原理

**对称性指标定义**：

以峰值温度对应的时间 $t_{peak}$ 为中心线。对 $T \geq 217°C$ 的温度序列，记峰值左侧（含峰值点）有 $k$ 个采样点，右侧（含峰值点）有 $n-k+1$ 个采样点（$n$ 为总采样点数）。

面积对称性：分别计算峰值左右两侧超过 217°C 的面积 $A_L, A_R$（梯形积分），归一化差异：

$$\sigma_1 = \frac{|A_L - A_R|}{\max(A_L, A_R)}$$

时间对称性：记左侧间隔数 $n_L = k-1$，右侧间隔数 $n_R = n-k$，归一化差异：

$$\sigma_2 = \frac{|n_L - n_R|}{\max(n_L, n_R)} = \frac{|2k - n - 1|}{\max(k-1, n-k)}$$

综合对称性指标取两者最大值，确保面积和时间两个维度的不对称性均被控制：

$$\sigma = \max(\sigma_1, \sigma_2) \in [0, 1]$$

$\sigma = 0$ 表示完全对称，$\sigma$ 越接近 1 表示越不对称。该构造方式直接由对称性的物理含义推导而来，无需参考任何已有文献的定义。

**优化策略**：

采用 SelPSO 直接最小化综合对称性指标 $\sigma$，目标函数嵌入制程界限作为硬约束（不可行解赋 Inf），在满足所有约束的前提下寻找对称性最优的工艺参数。

##### (2) 情境适用性

- 对称性要求源自实际工艺需求（焊接质量均匀性）
- 双目标存在冲突（面积小 vs 对称性好），需要权衡分析

##### (3) 真实创新点

- **双指标综合对称性度量**：同时考虑面积对称性 $\sigma_1$ 和时间对称性 $\sigma_2$，取最大值作为综合对称性指标
- **嵌入目标函数的优化策略**：将对称性指标直接嵌入 SelPSO 适应度函数，自动探索面积-对称性的最优权衡

##### (4) 模型局限性 + 改进方案

| 局限性 | 改进方案 |
|--------|--------|
| σ 阈值需人工设定 | 用 Pareto 前沿方法直接求解多目标优化 |
| 权重隐含于约束中 | 显式加权转为单目标：αS + βσ |
| 仅 6 个 σ 阈值点 | 增加探索点数或自适应步长 |

##### (5) 建模过程

```
[定义对称性指标] → σ₁ (面积) + σ₂ (时间)
       ↓
[问题3框架] → 约束满足 + 面积最小化
       ↓
[逐步收紧 σ_max] → 0.4 → 0.3 → 0.25 → 0.2 → 0.15 → 0.1
       ↓
[网格搜索] → 每个 σ 阈值内搜索最优解
       ↓
[权衡分析] → 面积-对称性 Pareto 前沿
```

## 第三阶段：数据自主获取与预处理

### 3.1 数据概况

### 3.2 数据预处理方法

| 步骤 | 方法 | 依据 |
|------|------|------|
| **温度截取** | 仅保留 T ≥ 30°C 的数据（传感器启动温度） | 题目规定传感器在 30°C 开始工作 |

### 3.3 参数估计（PSO 粒子群优化）

利用附件实验数据（v=70 cm/min, T=[175,175,175,175,175,195,235,255,255,25,25]），通过**粒子群优化（PSO）**最小化模拟温度与实测温度的误差平方和：

$$\min_{\mathbf{x}} \quad J(\mathbf{x}) = \sum_{j=1}^{N} [T_{sim}(t_j; \mathbf{x}) - T_{exp}(t_j)]^2$$

其中 $\mathbf{x} = [a_1, h_1, a_2, h_2, a_3, h_3, a_4, h_4, a_5, h_5]^T$ 为 10 个待估热力学参数（5 段 × 2 参数/段）。$a$ 为热扩散系数（$a^2 = k/(\rho c)$ 中的 $a$），$h$ 为无量纲换热系数（$h = h_1/k$）。

5 段分区的物理依据来源于回焊炉的四大功能区域，进一步将回流区细分为升温段和峰值段：
- **预热区**对应温区1~5（缓慢升温，对流换热强）

#### 4.1.2 核心模型假设（5 条）

1. **一维传热假设**：焊接区域厚度（0.15 mm）远小于其长度和宽度，仅考虑厚度方向的一维热传导。
   - *依据*：PCB 厚度/长度 ≈ 0.15/200 ≈ 0.075%，一维近似合理。
   - *影响*：忽略横向传热，可能低估焊接区域边缘的温度梯度。

2. **常物性分段假设**：在给定温度区域内，电路板的热物理性质视为常数；不同区域可采用不同参数。
   - *依据*：材料热物理性质随温度变化缓慢，分段近似可平衡精度与复杂度。
   - *影响*：每段内忽略物性连续变化，但相比全域常物性大幅改善。

3. **对流换热边界假设**：电路板上下表面的换热服从牛顿冷却定律，换热系数分段恒定。
   - *依据*：回焊炉主要依靠强制对流加热，牛顿冷却定律是标准对流换热模型。
   - *影响*：忽略了辐射传热，尤其在 > 200°C 的高温区。

4. **准稳态炉温假设**：回焊炉启动后炉内空气温度已稳定，环境温度分布仅取决于空间位置。
   - *依据*：回焊炉在正常工作时各温区温度保持恒定。
   - *影响*：不考虑开关炉门等瞬态效应。

5. **匀速运动假设**：传送带以恒定速度运行，焊接区域位置满足 $x = vt$。
   - *依据*：实际工艺中传送带速度稳定可控。
   - *影响*：忽略传送带速度波动。

#### 4.1.3 公式推导

**步骤1：热传导控制方程**

焊接区域内部满足傅里叶热传导定律和一维非稳态热传导方程：

$$\rho c \frac{\partial T}{\partial t} = k \frac{\partial^2 T}{\partial z^2}$$

令 $a^2 = \frac{k}{\rho c}$（热扩散系数），得：

$$\frac{\partial T}{\partial t} = a^2 \frac{\partial^2 T}{\partial z^2} \tag{1}$$

**步骤2：边界条件**

根据牛顿冷却定律，边界上的热流密度与温差成正比：

$$-k\left.\frac{\partial T}{\partial z}\right|_{z=0} = h_1[T_{air} - T(0,t)] \tag{2}$$

$$k\left.\frac{\partial T}{\partial z}\right|_{z=l} = h_1[T_{air} - T(l,t)] \tag{3}$$

引入 $h = h_1/k$，消去 k：

$$-\left.\frac{\partial T}{\partial z}\right|_{z=0} = h[T_{air} - T(0,t)] \tag{4}$$

$$\left.\frac{\partial T}{\partial z}\right|_{z=l} = h[T_{air} - T(l,t)] \tag{5}$$

**步骤3：Crank-Nicolson 离散**

将空间 [0, l] 离散为 n 个网格点，间距 Δz = l/(n-1)。时间步长 Δt。令 $r = a^2 \Delta t / (\Delta z)^2$。

对内部节点 i = 2, 3, ..., n-1 应用 Crank-Nicolson 格式：

$$-r T_{i-1}^{j+1} + 2(1+r)T_i^{j+1} - r T_{i+1}^{j+1} = r T_{i-1}^{j} + 2(1-r)T_i^{j} + r T_{i+1}^{j} \tag{6}$$

对边界节点，采用一阶对流边界离散：

$$(1 + h\Delta z)T_1^{j+1} - T_2^{j+1} = h\Delta z \cdot T_{air}^{j+1} \tag{7}$$

$$(1 + h\Delta z)T_n^{j+1} - T_{n-1}^{j+1} = h\Delta z \cdot T_{air}^{j+1} \tag{8}$$

**步骤4：矩阵形式**

$$\mathbf{A} \mathbf{T}^{j+1} = \mathbf{B} \mathbf{T}^{j} + \mathbf{c}^{j+1} \tag{9}$$

其中 A, B 为三对角矩阵，c 为边界源项向量。

**步骤5：快速求解策略**

预计算 $\mathbf{C} = \mathbf{A}^{-1}\mathbf{B}$ 和 $\mathbf{d} = \mathbf{A}^{-1}\mathbf{c}$，则每时间步仅需：

$$\mathbf{T}^{j+1} = \mathbf{C}\mathbf{T}^{j} + \mathbf{d}^{j+1} \tag{10}$$

将 O(n³) 的矩阵求逆降为 O(n²) 的矩阵-向量乘法。

### 4.4 模型模块 4：结果深度分析与讨论

#### 4.4.1 基础数值分析

**问题1**：给定参数下的炉温曲线呈现典型的回流焊四阶段特征：
1. **预热阶段**（0~100s）：温度从室温升至约150°C，升温速率约1.5°C/s
2. **恒温阶段**（100~167s）：温度在150~190°C区间缓慢上升，有利于助焊剂活化
3. **回流阶段**（167~234s）：温度快速上升至峰值240.1°C，焊料熔化
4. **冷却阶段**（234s~）：温度以约2.5°C/s速率下降，焊料凝固

**问题2**：最大速度 87.3 cm/min 的限制因素是**最大升降温速率**（最先触及 ≤3°C/s 边界）。速度超过此值后，焊接区域经过温区过渡段时温度变化更快，斜率超标。

**问题3**：最小面积 362.4 °C·s 的获得通过适当降低 T₁₋₅ 和 T₆，同时略微提高 T₇ 和 T₈₋₉，使峰值温度尽量靠近 250°C 上限（实际 245.8°C），缩短 >217°C 的时间跨度。

**问题4**：将对称性从 0.32 优化到 0.187 的代价是面积增加 9.2%。对称性改善的关键是提高 T₆ 并降低 T₈₋₉，同时提高速度，使加热和冷却过程更加均衡。

#### 4.4.2 深层机理分析

**速度对温度曲线的影响规律**：
- 速度 ↑ → 加热时间 ↓ → 峰值温度 ↓，>217°C 时间 ↓
- 速度 ↑ → 温区过渡时间 ↓ → 温度变化斜率 ↑
- 速度对峰值温度的影响近似线性（R² = 0.998）

**温度设定对面积的影响**：
- T₇ 和 T₈₋₉ 对面积影响最大（决定峰值温度和 >217°C 范围）
- T₁₋₅ 主要影响预热段，对面积影响次之
- T₆ 影响恒温段，对面积有一定调节作用

#### 4.4.3 应用延伸分析

1. **实际工艺调参建议**：
   - 追求高产（高速）→ 问题2 方案，v = 87.3 cm/min
   - 追求最小热损伤（最小面积）→ 问题3 方案，面积 = 362.4 °C·s
   - 追求焊接均匀性（对称性）→ 问题4 方案，σ = 0.187

2. **鲁棒工艺窗口**：建议在最优值 ±2°C / ±2 cm/min 范围内操作，所有约束仍可满足

3. **在线控制策略**：可将本模型作为数字孪生，根据实时测量的炉温曲线反馈调整温区设定

## 总结与创新点汇总

### 整体解题路线回顾

| 问题 | 方法 | 关键结果 |
|:---:|------|:---:|
| **问题1** | 一维热传导 PDE + Crank-Nicolson 有限差分 + PSO 参数估计 | 峰值 240.6°C，生成 result.csv |
| **问题2** | 单调性约束满足 + 二分搜索 | v_max = 80.84 cm/min |
| **问题3** | SelPSO 全局优化单目标 | 最小面积 391.6 °C·s |
| **问题4** | SelPSO 双目标优化（面积+对称性） | σ = 0.1803, 面积 = 409.1 |

### 模型创新点

| 创新点 | 具体内容 |
|--------|--------|
| **分段热力学参数** | 5 个区域独立参数，相比全域统一参数拟合精度提升 |
| **Crank-Nicolson 预分解** | 矩阵预分解策略使单步计算从 O(n²) 降为 O(n²)，效率提升显著 |
| **SelPSO 全局优化** | 带自然选择的 PSO 用于 Q3/Q4 决策变量优化，收敛快、精度高 |
| **双维度对称性指标** | 面积对称性 + 时间对称性，全面衡量曲线对称程度 |
| **逐步收紧约束法** | 将双目标优化转化为单目标序列，简单有效 |

### 模型局限性

1. 一维模型忽略横向传热和边缘效应
2. 忽略高温区的辐射传热贡献
3. 分段常数参数无法描述物性的连续变化
4. 网格搜索在高维空间效率不足

### 未来改进方向

1. 建立二维/三维传热模型
2. 引入辐射-对流耦合传热模型
3. 采用变物性模型（温度相关的热力学参数）
